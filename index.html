SemanticKO



Description


SemanticKO is a JavaScript library for building complex JavaScript applications in a declarative way on top of a RDF data layer. The library is open source and can be downloaded from github.

SemanticKO is built on top of KnockoutJS, a JavaScript library for building applications using the Model View ViewModel (MVVM) architectural pattern. It also uses RDFStore-JS a pure JavaScript of a RDF store as the foundation of the data layer.

A detailed description of the library can be found here.
